    这是我初次学习神经网络和深度学习不到一个月的作品。

    这只是简单的运用了两个卷积层，两个relu层，两个池化层和一个全联接层的、非常非常简单
且基础的神经网络模型。这个作品虽然简陋，但是也耗费了不少时间。由于理论方面和技术方面的青
涩和不熟练，导致各种问题可谓是接踵而至。从gpu的使用，到全联接层的参数问题，甚至是网络的
搭建都给我带来了不小的麻烦。感谢B站，以及梅爷，终于在跌跌撞撞中算是搞出了一个雏形。

    在这个CNN网络中，运用了gpu技术，同时为了保证学习运行的效率选择在AutoDL上进行远程
训练，因此会看到测试集、训练集路径不对的问题。
    在进行了三十轮训练后，在训练集上的正确率约为52%，在测试集上的正确率约为98%。同时，在
对训练过程中的训练正确率进行分析时发现，训练集上的正确率显著低于测试集，预计问题是该神经
网络比较简陋，因此出现了欠拟合问题。
    改进思路：1）尝试加入tensorboard可视化工具；
            2）尝试进行更复杂的神经网络，例如resnet；
            3）阅读论文，学习一些模型，并运用到程序中；
            4）欠拟合问题已经出现，应当防备过拟合问题，因此应当学习预防过拟合的方法。

    以上是我对这个简易甚至是有点简陋的神经网络模型的一点反思，其实现在还是有点窃喜，甚至
小小的兴奋，毕竟这可是我亲手搭建的第一个神经网络深度学习模块啊！
